Patronage — Render Reference

Il Popolo

The decorative citizen figures that wander plazas, roads and markets — five procedural silhouettes, a two-tone Renaissance cloth palette, and the marble recolor used when a figure is repurposed as a commissioned statue. Rendered here as flat specimen icons standing in for the low-poly, flat-shaded Babylon meshes in app/game/render/citizenFigures.ts.

Materials

Every figure is painted from three slots — robe, accent, skin — mapped onto shared submaterials at clone time so no citizen carries its own material.

Accent — hoods, hats, capes, trim

The Five Silhouettes

Each citizen clones one of five variant builders. Every one carries a front/back asymmetry so it reads directionally when it turns to face its walk. Captured live from the Babylon scene (dev build, tan robe / wine accent, fixed camera) — not a redrawn icon.

Colorways

On spawn, each figure gets a robe color and a differing accent color, redrawn here per variant using the game's own two-tone pairing.

Statue Recolor

For a commissioned Work, createStatueMesh reuses the same five silhouettes but collapses all three material slots onto one marble material — the citizen becomes a monument. Which variant a given Work gets is picked by hashing its artwork id (hash(artworkId) % 5), then set on the stone plinth from displayArt.ts.

Source: app/game/render/citizenFigures.ts (silhouettes, palette, statue collapse), app/game/render/citizens.ts (spawn & walk), app/game/render/displayArt.ts (plinth + marble material). Icons on this page are stylized 2D stand-ins, not exports of the actual meshes.
